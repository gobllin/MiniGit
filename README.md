# MiniGit – Lightweight Version Control System in C

MiniGit is a simplified Git-like version control system implemented in C to explore the internal architecture and working principles of distributed version control systems. The project focuses on low-level filesystem operations, repository management, and content hashing through a custom command-line interface.

## Implemented Features

### Repository Management
- Repository initialization
- Internal repository structure setup
- Filesystem organization for version tracking

### Hashing System
- Content-based hashing utilities
- Object identification mechanisms
- Hash generation for file tracking

## Tech Stack
- C Programming Language
- File Handling
- Filesystem Operations
- Hashing Techniques
- Command-Line Interface (CLI)

## Learning Goals
This project was built to understand:
- Internal Git architecture
- Version control system fundamentals
- Content-addressable storage concepts
- Low-level systems programming in C
- Filesystem-based object management
